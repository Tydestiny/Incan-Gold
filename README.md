# 在线印加宝藏游戏 (Incan-Gold)

这是一个基于Web的多人在线印加宝藏游戏实现，使用Node.js、Socket.IO和原生JavaScript开发。

## 功能特性
- 支持2-10人在线游戏
- 实时游戏状态同步
- 房主创建房间并开始游戏
- 玩家可以选择继续探险或返回营地
- 自动处理游戏逻辑（回合、卡牌翻开、宝藏分配等）

## 安装和运行
1. 确保已安装Node.js
2. 在项目根目录下运行 `npm install` 安装依赖
3. 运行 `node server.js` 启动服务器
4. 在浏览器中打开 `http://localhost:3000` 访问游戏

## 代码结构
- `server.js`: 服务器主文件，处理Socket.IO连接和游戏房间管理
- `game_logic/game.js`: 游戏核心逻辑实现
- `public/`: 前端文件目录
  - `index.html`: 游戏主页面
  - `client.js`: 客户端JavaScript代码
  - `style.css`: 游戏样式表

## 游戏规则简述
- 游戏进行5个回合
- 每回合玩家选择继续探险或返回营地
- 探险过程中会翻开卡牌（宝藏或危险）
- 发现宝藏时，返回营地的玩家可以平分宝藏
- 遇到相同危险卡两次时，当前回合结束，所有探险者失去本轮宝藏

## 开发说明
- 服务器使用Socket.IO实现实时通信
- 游戏逻辑封装在`IncanGoldGame`类中
- 前端通过Socket.IO与服务器通信，更新UI
